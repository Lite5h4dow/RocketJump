# Rocket Jump

### Game Mechanics
 - Main Menu
 - Normal Jump for Low / Medium Objects
 - Super Jumps for Medium / High objects & Bonusses
 - Duck for medium / High Objects
 - Scoring for coins and time
 - Play / Pause Menu
 - Adds
 - Boost Pickups
 - Share Score Feature

### Game Properties
1. Simple Mobile Game.
2. Playable with one hand.
3. "On The Shitter" Game.
