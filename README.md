# Rocket Jump

### Game Mechanics
 - Main Menu
 - Normal Jump chains into Rocket Jump that you can control [Done.]
 - Duck for medium / High Objects
 - Scoring for coins and time
 - Play / Pause Menu
 - Ads
 - Boost Pickups
 - Share Score Feature

### Game Properties
1. Simple Mobile Game.
2. Playable with one hand.
3. "On The Shitter" Game.

### Obstacles
 - Flock of Birds
 - Hot Air Baloon
 - Airliners
 - cars
